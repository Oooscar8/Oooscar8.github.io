工程问题建模与仿真案例2-代码说明

520021911048 唐波昊

代码文件夹介绍

  • 代码文件夹中包含4个文件夹，1个readme文档。其中 basic 代表基础要求部分，expand1，expand2，expand3，expand4 代表拓展1，2，3，4部分。

basic文件夹

  • main.m 文件：基础任务主函数。
  • A.m 文件：用符合负指数分布随机数，随机生成切换器 A 的寿命和状态。
  • B.m 文件：用符合负指数分布随机数，随机生成切换器 B 的寿命和状态。
  • Node.m 文件：输入切换器 A 和 B 的状态，得到 A 和 B 所在节点的状态。
  • System_condition.m 文件：利用输入的所有 n 个节点的状态，得到系统当前的状态。
  • System_life.m 文件：将调用前面 4 个函数文件，对于输入的 n，随机生成一个系统，并得到这个随机系统的寿命。

expand1文件夹

  • KeYong.m 文件：生成可用性（拓展1任务主函数）。
  • P_keyong.m 文件：生成不同情况的概率。

expand2文件夹

  • time_bianbuchang.m 文件：生成变步长方法运行时间（拓展2任务主函数1）。
  • time_dengbuchang.m 文件：生成等步长方法运行时间（拓展2任务主函数2）。
  • A.m 文件：用符合负指数分布随机数，随机生成切换器 A 的寿命和状态。
  • B.m 文件：用符合负指数分布随机数，随机生成切换器 B 的寿命和状态。
  • Node.m 文件：输入切换器 A 和 B 的状态，得到 A 和 B 所在节点的状态。
  • System_condition.m 文件：利用输入的所有 n 个节点的状态，得到系统当前的状态。
  • System_life.m 文件：将调用前面 4 个函数文件，对于输入的 n，随机生成一个系统，并得到这个随机系统的寿命。

expand3文件夹

  • main.m 文件：拓展3主函数。
  • A.m 文件：用符合负指数分布随机数，随机生成切换器 A 的寿命和状态。
  • B.m 文件：用符合负指数分布随机数，随机生成切换器 B 的寿命和状态。
  • Node.m 文件：输入切换器 A 和 B 的状态，得到 A 和 B 所在节点的状态。
  • System_condition.m 文件：利用输入的所有 n 个节点的状态，得到系统当前的状态。
  • System_life.m 文件：将调用前面 4 个函数文件，对于输入的 n，随机生成一个系统，并得到这个随机系统的寿命。

expand4文件夹

  • newmain.m 文件：拓展4任务主函数。
  • newA.m 文件：用符合负指数分布随机数，随机生成切换器 A 的寿命和状态。
  • new’B.m 文件：用符合负指数分布随机数，随机生成切换器 B 的寿命和状态。
  • newNode.m 文件：输入切换器 A 和 B 的状态，得到 A 和 B 所在节点的状态。
  • newSystem_condition.m 文件：利用输入的所有 n 个节点的状态，得到系统当前的状态。
  • newSystem_life.m 文件：将调用前面 4 个函数文件，对于输入的 n，随机生成一个系统，并得到这个随机系统的寿命。
  • draw1.m 文件：绘制原始方案和改进方案的MTTF对比曲线。
  • draw2.m 文件：绘制原始方案和改进方案的系统可靠性对比曲线。
  • draw3.m 文件：绘制原始方案和改进方案的阻塞故障占比对比曲线。

运行使用说明

  • 基础任务 ：运行 main.m文件。
  • 拓展1 ：运行 KeYong.m 文件。
  • 拓展2 ：运行 time_bianbuchang.m，time_dengbuchang.m文件。
  • 拓展3 ：运行 main.m 文件。
  • 拓展4 ：运行newmain.m，draw1.m，draw2.m，draw3.m 文件。

Machine Learning —— By 李宏毅

学习顺序

  • 讲义 (41h)

    • 第1讲 Introduction (17h)
    • 第1讲：深度学习介绍：主要是基础概念的介绍，快速过一遍。

  1. 机器学习基础概念：p1-p2-p3-p7 (3h)

  2. 我是土堆10h Pytorch速成视频 (10h)

  3. 反向传播基础知识：p8 (31m)

  4. 案例讲解回归问题：p9 (78m)

  5. 案例讲解分类问题: p10-p11 (137m)

    • 第2讲 What to do if my network fails to train (4h)
    • 第2讲：为什么训练网络会失败：主要是将训练网络的一些细节，局部最小值，鞍点，自适应学习率，损失函数等。这一讲的选修的梯度下降必看，新的优化器可以先不看，如果有余力可以看，主要讲了对梯度下降的一些改进。

  6. Classification: p12-p19 (重点是p18-p19梯度下降算法) (250m)

    • 第3讲Image as Input (2.5h)
    • 第3讲：图像作为输入：CNN网络，必看，非常重要

  7. 卷积神经网络:p22-p25 (140m)

    • 第4讲 Sequence as input (3.5h)
    • 第4讲：序列作为输入：先看选修的RNN，再去看自注意力机制，不要搞错顺序。因为注意力太火了，所以RNN放在了选修，不过我认为还是要先看RNN模型基础，再去看自注意力机制，为下面额Transformer模型做准备。选修中的GNN网络，可以你自己的需求，入门阶段可以先跳过不看。

  8. RNN: p28-p29 (140m)

  9. 自注意力机制: p26-p27 (75m)

    • 第5讲 Sequence to sequence (3h)
    • 第5讲：序列到序列：主要讲了Transformer模型，必看，选修的指针网络可以先不看。

  10. Transformer: p32-p34 （165m)

    • 第6讲 Generation (4h)
    • 第6讲：生成模型：主要是对GAN理论的介绍。看你自己研究方向，如果是GAN方向的，可以细细看下，如果入门选手，直接跳过。

  11. GAN + VAE: p37-p40、p44 (225m)

    • 第7讲 Self-supervised learning (4h)
    • 第7讲：自监督学习，必看，很火。主要看关于BERT介绍相关的视频，比如模型介绍，微调，预训练等。BERT的各种变体比如Spanbert等可以先不看，同理GPT有余力就看，没余力直接跳过，不影响。

  12. Self-supervised learning + BERT + GPT-3: p46-p52（250m)

    • 第8讲 Auto-encoder (1h)

  13. Auto-encoder: p53-p54 (50m)

    • 第10讲 Attack (1.5h)

  14. Attack: p59-p60 (75m)
